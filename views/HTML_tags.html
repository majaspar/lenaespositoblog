Let's take a look at the less popular HTML tags.

There are numerous HTML tags that we do not need for every single page that we work on. Of course we can always use a div which with some styling can replace just about anything, but it's good to remember that div has no semantic meaning, so each time we use it, the browser understands our code a little bit less.

The <pre> tag defines preformatted text. Text in a <pre> element is displayed in a fixed-width font, and the text preserves both spaces and line breaks. The text will be displayed exactly as written in the HTML source code.
